## ğŸ“– Getting Started

### Prerequisites

Make sure you have the following installed:
- **Node.js** (LTS version)
- npm or yarn package manager
- (Optional) **Database tool** if using a database engine like MongoDB or PostgreSQL.

### Setup Instructions

1. **Clone the Backend Repository**:
   ```bash
   git clone https://github.com/shauryaverma03/GearGuard-The-Ultimate-Maintenance-Tracker.git
   ```
2. **Navigate to the Backend Folder**:
   ```bash
   cd GearGuard-The-Ultimate-Maintenance-Tracker/backend
   ```
3. **Install Dependencies**:
   ```bash
   npm install
   ```
4. Set up environment variables (e.g., `.env`) to include secrets like:
   ```plaintext
   DB_CONNECTION=your_database_url
   JWT_SECRET=your_jwt_secret
   ```
5. **Run the Server**:
   ```bash
   npm run dev
   ```

   Your backend server will be running at `http://localhost:5000/`.
